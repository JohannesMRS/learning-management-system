# VeloLearn

**VeloLearn** adalah platform Learning Management System berbasis website yang dirancang untuk memudahkan interaksi antara pengajar dan siswa dalam proses belajar-mengajar secara daring


## Teknologi Yang Digunakan

*   **Backend & Framework:** ![Laravel](https://shields.io) (Versi 10/11)
*   **Database:** ![MySQL](https://shields.io) 
*   **Frontend Styling:** ![TailwindCSS](https://shields.io)
*   **Interaktivitas:** ![JavaScript](https://shields.io) (Vanilla JS)
